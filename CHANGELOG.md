### 🟢 指定发送形式功能

支持自定义歌曲发送格式，包括卡片、语音、文件、文本等多种格式，满足不同场景下的发送需求。

**使用方式**:

当唤出歌曲列表后,输入<列表序号> <模式> 即可选择指定模式发送歌曲

例如：

`1 卡片`  -> 发送列表第一首歌曲为卡片格式

`2 语音`  -> 发送列表第二首歌曲为语音格式

`3 文件`  -> 发送列表第三首歌曲为文件格式

`4 文本`  -> 发送列表第四首歌曲为文本格式

`1 1` -> 发送列表第一首歌曲为卡片格式

`2 2` -> 发送列表第二首歌曲为语音格式

`3 3` -> 发送列表第三首歌曲为文件格式

`4 4` -> 发送列表第四首歌曲为文本格式

`1 card` -> 发送列表第一首歌曲为卡片格式

`2 record` -> 发送列表第二首歌曲为语音格式

`3 file` -> 发送列表第三首歌曲为文件格式

`4 text` -> 发送列表第四首歌曲为文本格式
